#  Greatness Portfolio Website

A modern and mobile-responsive portfolio website built with pure HTML and CSS.

This portfolio was designed for **Greatness** and features a clean UI, responsive layout, smooth navigation, and modern styling — without using any images.

---

#  Features

- Responsive design for mobile and desktop
- Modern dark-themed UI
- Sticky navigation bar
- Hero section
- About section
- Skills section
- Projects section
- Contact section
- Smooth scrolling
- Clean and beginner-friendly code
